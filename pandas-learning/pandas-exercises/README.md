# Pandas Exercises

This folder contains Python exercises to practice **Pandas** functionality using custom CSV datasets.

## Exercises Overview

1. **Exercise 1** – Basic DataFrame creation and exploration:
   - Creating DataFrames from dictionaries
   - Viewing first rows (`head`)
   - Getting info and summary statistics (`info`, `describe`)
   - Inspecting columns and indices

2. **Exercise 2** – Working with CSVs (`employee.csv`):
   - Loading CSV data into DataFrames
   - Selecting rows and columns using `.loc` and `.iloc`
   - Boolean filtering

3. **Exercise 3** – Handling missing values (`product.csv`):
   - Filling missing values with mean, zero, or placeholder
   - Dropping rows or columns with null values
   - Replacing specific values

4. **Exercise 4** – Grouping and sorting (`sales.csv`):
   - Grouping data using `groupby` and aggregation functions
   - Sorting values

5. **Exercise 5** – Merging DataFrames (`employees.csv` & `salaries.csv`):
   - Performing different types of joins: `left`, `inner`, `right`, `outer`

## CSV Files

- `employee.csv`
- `product.csv`
- `sales.csv`
- `employees.csv`
- `salaries.csv`

All CSVs are **custom datasets** created for practice purposes.


These exercises are part of learning and strengthening **Pandas skills** for data manipulation, analysis, and joining datasets.
